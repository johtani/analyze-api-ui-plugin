# analyze-api-ui-plugin

> UI for elasticsearch analyze API

Analyze text with Analyzer
![Sample image](docs/sample_image.jpg)

This branch is for Kibana 7.x.

## How to use?

See [Getting Started](docs/GETTING_STARTED.md)
